Steps for creating sockets:

Create directory ~/Brackets/sockets
Run git init to initialize the git repository
Run npm init to initialize npm and create the package.json file
Run git add . to add the package.json file
Run git commit to do your first commit
Go to github.com and create a new repository
Run the code from github on the terminal
Push the code up to github
Run npm install express@4.13.3. --save to create a new server
Run heroku create to create a new heroku site
Run git push heroku to push the files up to the new heroku site
Run heroku open to open the site in your browser

Run npm install socket.io@1.3.7 --save to install sockets
